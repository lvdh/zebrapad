# zebrapad

Label printing tool for ZPL-based label printers.

## Usage

- `make test` to run tests
- `make run` to start the server (default: http://localhost:8080)
- `make clean` to remove build artifacts

## Directory Structure

- `src/` - backend
- `src/web/` - frontend (HTML, CSS, Bulma)
- `bin/` - build artifacts
